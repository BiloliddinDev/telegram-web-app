# Contributing

Loyihaga hissa qo'shish uchun quyidagi qadamlarni bajaring:

## Development Setup

1. Repo'ni fork qiling
2. Clone qiling: `git clone https://github.com/your-username/telegram-webapp.git`
3. Branch yarating: `git checkout -b feature/your-feature`
4. O'zgarishlarni qiling
5. Commit qiling: `git commit -m "Add your feature"`
6. Push qiling: `git push origin feature/your-feature`
7. Pull Request yarating

## Code Style

- Backend: ESLint qoidalariga rioya qiling
- Frontend: Prettier va ESLint qoidalariga rioya qiling
- TypeScript: Type safety'ga e'tibor bering
- Comments: Muhim joylarni comment qiling

## Testing

- Backend API'larni test qiling
- Frontend komponentlarni test qiling
- Integration testlar yozing

## Pull Request

- Clear description yozing
- Screenshots qo'shing (agar UI o'zgarishi bo'lsa)
- Related issues'ni mention qiling
